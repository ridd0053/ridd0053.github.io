Consultancy organisaties:

Persoonlijk vond ik de consultancy organisaties vriendelijk, maar ook formeel. Naar mijn mening komen de bedrijven erg grootschalig over. Zelf zie ik mij liever werken in een kleinere organisatie.
Consultancy bedrijven geven natuurlijk advies aan een derde partij. Ondanks dat het mij leuk lijkt om mijn kennis te delen met andere, lijkt het mij nog erg eng om advies te geven aan andere.
Dit kan natuurlijk veranderen naarmate ik mijn kennis opbouw.

Als functies kwam ik bij data & analytics onder andere Data engineer tegen.
Als technieken zag ik daar SQL, azure en agile/devops. Onder data & analytics zag ik ook de functie Data consultant.
Hierbij moet je ook kennis hebben van SQL. Als laatste zag ik nog Data science bij data & analytics. Hier zag ik dat je kennis moet hebben van de scripttaal Python. Bij de functie cybersecurity zag ik dat je kennis moest hebben van Cloud security, DevSecOps, IoT security, Networks security en Endpoint security. Veder zag ik ook de functie Software development met de technieken Java, PHP, C# en .NET. Java zou je niet enkel nodig hebben voor Software development, maar ook voor Software engineer. Echter zou je ook kennis mogen hebben van soortgelijke zoals Python en C#. Als software engineer moet je ook kennis hebben van Object oriented programming. Verder moet je nog kennis hebben van git, scrum en de programmeertaal SQL.

Software organisaties:

De software organisaties vond ik informeel en ik vond sfeer fijn. Ik dacht dat ik projectmatig werken heel leuk zou vinden, maar een software product maken lijk mij nog leuker. Het leuke aan een product maken is dat je het nog constant kan verbeteren. Het lijkt mij ook leuk dat je als team aan een product werkt, maar wel een specialisatie hebt in het verloop.

Een functie die ik bijvoorbeeld tegen kwam was Java-Developer. Als Java-Developer heb je onder andere kennis nodig van Spring, Typescript en Angular. Voor de functie Front-end developer heb je ook kennis nodig van Angular. Qua programmeertalen heb je kennis nodig van Java en C#. Verder technieken zijn HTML, CSS, Docker en nog vele andere. Je hebt natuurlijk kennis nodig van .NET bij de functie .NET developer. Als programmeertaal kwam ik bij deze functie C# tegen. C# heb je ook nodig bij de functie Software development. Voor de functie Platform Engineer heb je onder andere ervaring nodig met Docker, Mysql en Git.

Financiële organisaties:

De financiële organisaties lijken mij erg formeel en grootschalig. De functies die betrekening hebben op engineering zijn Front-end engineer, Software engineer en Linux engineer. Verder kwam ik ook nog de functies Sharepoint-specialist en Java Developer tegen. Er wordt gebruik gemaakt van de opmaaktalen HTML5 en CSS. Qua programmeertalen wordt er gebruik gemaakt van Java en als scripttaal wordt Javascript gebruikt. Bij de functie Linux engineer wordt er gebruik gemaakt van Linux en bij andere functies wordt er gebruik gemaakt van Angular.

Overheidsorganisaties:

Naar mijn mening hadden de overheidsinstanties wel een nette cultuur, want ze waren wel netjes gekleed. Overheidsinstanties zijn niet commerciële ingesteld en ik voel mij aangetrokken tot deze instelling.
Je kan vrij veel verschillende ICT-functies uitvoeren bij overheidsorganisatie.

Zo heb je de functie software engineer, je zou dan als programmeertalen Python, Java en C# moeten kennen.
Als scripttalen zou je weet moeten hebben van Javascript of Typescript. Je zou verder ook bekend moeten zijn met het framework Scrum.
Bij de functie front-end developer moet je natuurlijk bekend zijn met HTML, CSS en Javascript. Veder moest je kennis hebben van Agile en ervaring hebben met het framework React.
Voor de functie .NET developer moet je natuurlijk kennis hebben van het framework .NET. Er werd ook kennis gevraagd naar ASP.NET. Als ontwikkelmethode dien je ervaring te hebben met TDD en qua programmeertalen dien je kennis te hebben van C#.
De programmeertaal Python heb je niet enkel nodig als software engineer, maar ook voor de functie Data Analist. Als Data Analist dien je ook weet te hebben van Splunk en Tableau. Splunk heb je ook nodig in de functie Cyber Security Analist.

Organisaties in de detailhandel:

De detailhandel organisaties zijn bezocht tijdens de bedrijvensafari. Hierdoor heb ik een beter beeld gekregen van de organisaties. De sfeer vond ik wisselvallig, dus het verschilde per bedrijf. De ene bedrijf vond ik gezellig en de ander vond ik erg strikt. Ik vond het wel leuk dat bij beide organisaties de kantoor en het magazijn zich in één ruimte bevond. Als je op kantoor werkt realiseer je soms niet altijd wat er allemaal in werkelijkheid gedaan wordt. Dat is dus hier niet het geval.

De technieken die bij de functie PHP developer horen zijn Javascript, PHP, SQL, HTML/CSS, Git en Docker.
Bij de functie javascript developer heb je ook kennis nodig van Git en Javascript. HTML, CSS en Javascript heb je ook nodig bij de functie Front End Developer. Dan is er de nog functie Java engineer en hierbij moet je bekend zijn met Spring. Voor de functie Backend Software engineer heb je ervaring nodig met SQL, Javascript en Typescript.
De organisaties maken voornamelijk veel webapplicaties. Dat kan je terug zien aan de functies en de technieken.
